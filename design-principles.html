Design Principles

Felix S. Guzman

Emphasis

www.spotify.com/do

Spotify's website exemplifies emphasis because the page is looking for the user to subscribe to their premium package and the first section that a user sees when he enters the page is a discount for the subscription. Below the user can see the benefits of the premium subscription, and after the benefits the page shows every subscription plan so you can choose the one you prefer. This page creates emphasis on the product that the Spotify website is offering.

Repetition

Family Search

Family Search website we can see the web design principle of repetition. This principle is manifested in this website in every different section/ topic of the page as we cannotice that first comes the big font size phrases, then the smaller text and then a butoon to redirect the user to that specific topic or section.

Contrast

Brigham Young University - Idaho

Brigham Young University - Idaho is a wonderful example of the principle of contrast. Contrast, in web design means to highlight important details in the website in order to get the attention of the user. It could be through size, color, or content. In this specific website we can see that the first image is much larger than all of the other ones and that the words on the image have different font sizes to contrast and get the user's attention and to highlight the main message and first impression that they want the user to get from the website.
